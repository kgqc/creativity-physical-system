# 清理记录

## 删除的目录和文件

- 删除 `server/`：Express App、Session、路由、SQLite、RunningHub Client、节点映射、Job Worker、Webhook、结果下载与恢复逻辑均不属于前端交付。
- 删除 `server-dist/`：已编译的服务端产物。
- 删除 `data/`：本地 SQLite、上传和输出运行数据。
- 删除 `scripts/`：`init-db.ts` 和 `inspect-runninghub-workflow.ts` 仅服务于本地后端。
- 删除 `tsconfig.server.json`：不再编译服务端。
- 删除旧 `src/api/assets.ts`、`jobs.ts`、`projects.ts`：它们是按领域分散的直接请求封装，已由统一 `ApiService` 替代。
- 删除旧后端测试结果文档 `TEST_RESULTS.md`。
